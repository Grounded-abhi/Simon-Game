# Simon Game 🎮

A simple implementation of the classic **Simon Game** built using **HTML, CSS, JavaScript, and jQuery**. This project was made to practice DOM manipulation, event handling, animations, and game logic using JavaScript.

## Demo

🔗 https://grounded-abhi.github.io/Simon-Game/

## What it does

- Generates a random color sequence
- Plays sounds and animations for each button
- Checks user input against the generated sequence
- Increases the level after every successful round
- Ends the game on a wrong input and allows you to restart

## Tech Stack

- HTML
- CSS
- JavaScript
- jQuery

## Getting Started

Clone the repository:

```bash
git clone https://github.com/Grounded-abhi/Simon-Game.git
```

Move into the project folder:

```bash
cd Simon-Game
```

Then open `index.html` in your browser.

## How to Play

1. Press any key to start.
2. Watch the sequence shown by the game.
3. Repeat the sequence by clicking the colored buttons.
4. Each new level adds one more color.
5. If you click the wrong button, the game ends. Press any key to start again.
